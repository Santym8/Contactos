# Calculadora Santiago Fierro

Aplicación de contactos desarrollada para la asignatura de Aplicaciones Móviles UTN

## Función Principal

Guardar nuevos contactos
Buscar contactos
Actualizar un contacto cuando tiene el mismo nombre
